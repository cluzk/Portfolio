# Maze Solver
**Recursion | Depth-First Search | Backtracking**

* **Algorithm:** Utilized a recursive **Depth-First Search (DFS)** to navigate complex 2D grids.
* **Backtracking:** Implemented a state-reversal mechanism that "unmarks" paths when a dead-end is reached, ensuring the solver eventually finds a valid route.
* **Memory:** Managed the call stack effectively to prevent overflows during deep recursion.
